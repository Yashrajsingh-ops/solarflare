# Solar Flare Forecasting API

Production-ready FastAPI backend for **solar flare forecasting and nowcasting** using
Aditya-L1 **SOLEXS** (soft X-ray) and **HEL1OS** (hard X-ray) instrument data.

---

## Tech Stack

| Layer       | Technology                          |
|-------------|-------------------------------------|
| Web         | FastAPI + Uvicorn                   |
| ML          | XGBoost + scikit-learn              |
| Data        | Pandas + NumPy                      |
| Database    | SQLite (dev) / PostgreSQL (prod)    |
| Validation  | Pydantic v2                         |

---

## Quick Start

### 1. Clone and install dependencies

```bash
cd backend
python -m venv .venv
source .venv/bin/activate        # Windows: .venv\Scripts\activate
pip install -r requirements.txt
```

### 2. (Optional) Train the ML model

```bash
python ml/train_model.py
```

> If you skip this step, the API uses a physics-inspired heuristic fallback
> until `ml/xgboost_model.pkl` and `ml/scaler.pkl` are present.

Place your labelled training CSV at `data/training_data.csv` with columns:
`timestamp`, `soft_xray_flux`, `hard_xray_flux`, `label` (0/1).

### 3. Start the server

```bash
uvicorn app.main:app --reload
```

The API is now running at **http://127.0.0.1:8000**.

## API Reference

Interactive Swagger docs: **http://127.0.0.1:8000/docs**  
ReDoc: **http://127.0.0.1:8000/redoc**

### `GET /health`
Liveness probe.

```json
{ "status": "healthy" }
```

---

### `POST /api/upload`
Upload a SOLEXS/HEL1OS CSV file.

**Form field:** `file` (multipart/form-data, `.csv`)

**Required CSV columns:**

```
timestamp,soft_xray_flux,hard_xray_flux
2025-01-01 10:00:00,23.5,4.2
```

**Response:**
```json
{ "message": "Dataset uploaded successfully", "rows_ingested": 15 }
```

---

### `POST /api/predict`
Real-time flare probability for a single flux reading.

**Request:**
```json
{ "soft_xray_flux": 70, "hard_xray_flux": 35 }
```

**Response:**
```json
{
  "flare_probability": 84.5,
  "risk_level": "CRITICAL",
  "prediction": "Solar Flare Imminent"
}
```

Risk levels:

| Probability | Level    |
|-------------|----------|
| 0–25 %      | LOW      |
| 25–50 %     | MEDIUM   |
| 50–75 %     | HIGH     |
| 75–100 %    | CRITICAL |

---

### `GET /api/dashboard`
Latest observation with risk summary.

```json
{
  "latest_soft_flux": 72.0,
  "latest_hard_flux": 38.0,
  "risk_level": "HIGH",
  "flare_probability": 81.0
}
```

---

### `GET /api/alerts`
Current alert status based on the most recent prediction.

```json
{
  "alert": true,
  "message": "High probability solar flare detected — 84.5% [CRITICAL]"
}
```

## Frontend Integration

The API is CORS-enabled for all origins (restrict in production).  
A React dashboard can call:

```js
// Predict
const res = await fetch("http://localhost:8000/api/predict", {
  method: "POST",
  headers: { "Content-Type": "application/json" },
  body: JSON.stringify({ soft_xray_flux: 70, hard_xray_flux: 35 }),
});
const data = await res.json();
// { flare_probability: 84.5, risk_level: "CRITICAL", prediction: "..." }
```
